[Python] PelicanとGithub Pagesで作成したブログを独自ドメインとHTTPS対応

Posted on 2019年1月3日 in Tech

Pythonインストール・設定がまだの場合は[Python] MacにPythonのローカル開発環境を構築を参照
PelicanとGithub Pagesでブログ作成がまだの方は[Python] PelicanとGithub Pagesで簡単ブログ作成を参照。

DNSレコード設定
toyscreation.jpのドメインはAWSのRoute53で管理しているため、 Route53にてDNSレコード設定を行います。 

Name: www  
Type: CNAME - Canonical name  
Value: morita-toyscreation.github.io

GitHub Pages設定 対象ブランチのSettingsにてGitHub Pages設定を行います。

Custom domain: www.toyscreation.jp
Enforce HTTPS: チェック

独自ドメインとHTTPS対応完了
ブラウザでサイトを確認
https://www.toyscreation.jp
